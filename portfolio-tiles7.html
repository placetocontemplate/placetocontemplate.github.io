Projects

Scalable CSS

A modern, scalable front-end demo. Using a full-width jumbotron header. Coded from scratch using CSS, HTML and JavaScript, as a way to familiarize myself with all the modern front-end methods. Mobile-friendly.

Section Dividers

Fun, layered CSS headers or section dividers. Designed to add excitement to a website and highlight ideas or represent the overall theme. Making these was a chance to use the “parallax” background effect, and to create more full-width options for modern layouts.

Simple Cart

A simple JavaScript shopping cart designed to sell multiple amounts of just one product, like a book. It adds the option to choose quantity and shipping type to a PayPal button. Upon checkout, it sends the result to PayPal without needing the full PayPal API. It fits in a sidebar, for instance on a blog, and is easy to style with a stylesheet.
